# open_test
It provides local machines with "open2ch" access.

## requierments

+ python3
+ flask
+ requests
+ lxml

## usage

1. Start the local server. `python server.py`
1. Then access to local API. For example: `0.0.0.0:5000/bbslist/main`

## Warining

These scripts are just for test. Don't deploy it.
